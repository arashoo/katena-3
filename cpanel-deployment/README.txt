# Katena3 Deployment for cPanel 
 
INSTALLATION STEPS: 
 
1. Upload all files to your web directory 
2. In cPanel Terminal, run: npm install 
3. Run: npm start 
4. Access your site and login with: 9220katena 
 
FILES INCLUDED: 
- index.html (main app) 
- assets/ (CSS, JS, images) 
- server.js (Node.js backend) 
- package.json (dependencies) 
- data/ (glass inventory)
